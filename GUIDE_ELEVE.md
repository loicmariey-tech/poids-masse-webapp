# Guide eleve

## Objectif
Verifier la relation **P = m x g** en tracant les points experimentaux et en comparant la regression lineaire a la droite theorique.

## Etapes
1. Remplis **Nom / Prenom / Classe**.
2. Choisis le **nombre de mesures**.
3. Choisis les **unites** (kg/g pour la masse, N/mN pour le poids).
4. Entre les valeurs dans le tableau.
5. Clique **Mettre a jour** pour tracer le graphique.
6. Observe la pente de la regression et compare a **9.81 N/kg**.
7. Exporte le PDF si besoin.

## Aide
- Si tu as saisi la masse en g, l application convertit automatiquement en kg.
- L ordonnee a l origine b doit etre proche de 0 si la proportionnalite est verifiee.
